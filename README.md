Follow these commands to run the project

In a first terminal run : 

Move to the right directory :
```shell
cd client/
```

Install the dependencies :
```shell
npm install
```

Start the client : 
```shell
npm run watch
```

In a second terminal run :

Move to the right directory :
```shell
cd server/
```

Install the dependencies :
```shell
npm install
```

Start the server : 
```shell
nodemon
```
